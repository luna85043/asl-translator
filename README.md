# asl-translator
Final project for Artificial Neural Networks and Deep Learning at DIS Spring 2026.

## Datasets
Run `./download-datasets.sh` in the root directory (`asl-translator/`) to create the `datasets/` directory and download all the datasets. We're currently using the following datasets:
- ASL-HG, containing 36,000 images representing A-Z and 0-9: https://www.sciencedirect.com/science/article/pii/S2352340926000454
- synthetic-asl-alphabet on Kaggle https://www.kaggle.com/datasets/lexset/synthetic-asl-alphabet
- synthetic-asl-numbers on Kaggle https://www.kaggle.com/datasets/lexset/synthetic-asl-numbers
